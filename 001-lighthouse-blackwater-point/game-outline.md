# Game #1: The Lighthouse at Blackwater Point

## Overview
A decommissioned lighthouse on a rocky shore. Something ancient lives beneath it. The keeper disappeared 30 years ago. Tonight a storm forces the player inside.

## Scenes (6)
1. **The Shore** — Rocky shoreline, storm, locked lighthouse door. Find brass key in tide pool.
2. **Ground Floor** — Entry hall, keeper's desk, dusty kitchen. Find matches, read note about foghorn.
3. **Keeper's Quarters** — Upstairs bedroom. Keeper's journal, rusted crank, silver anchor charm.
4. **The Lamp Room** — Top of tower. Install crank, light lamp. See the creature in the water below.
5. **The Cave Beneath** — Underground cave under lighthouse. Use foghorn + anchor charm to seal creature.
6. **Dawn** (win scene) — Storm breaks. Light shines. Rescued.

## Items (5)
- **brass-key** — Opens lighthouse door. Found in tide pool crevice (scene-1).
- **matches** — Light the lamp. Found in kitchen drawer (scene-2).
- **keeper-journal** — Lore item. Reveals foghorn ritual. Found in bedroom (scene-3).
- **rusted-crank** — Powers the lamp mechanism. Found under bed (scene-3).
- **silver-anchor-charm** — Sealing artifact. Found in hidden compartment (scene-3).

## Puzzles (4)
1. **unlock-door** (scene-1): Use brass key on locked door → enter lighthouse
2. **light-the-lamp** (scene-4): Install rusted-crank, use matches on lamp → see creature
3. **foghorn-ritual** (scene-5): Combine foghorn + silver-anchor-charm at ritual circle → seal creature
4. **win-condition** (scene-5): Creature sealed → dawn breaks

## Win Condition
Seal the ancient creature in the cave using the foghorn ritual with the silver anchor charm. The storm breaks. Dawn arrives. The player is rescued.
